# Station Check POC

A minimal Blazor Server application demonstrating:
1. QR Code generation for stations
2. Location detection using browser's Geolocation API

## Features

- Generate QR codes for stations
- Get current location (latitude, longitude, accuracy)
- Simple UI with Bootstrap-like styling

## Running the Project

1. Make sure you have .NET 7.0 SDK installed
2. Clone this repository
3. Run the project:
```
dotnet run
```
4. Open browser at https://localhost:5001

## Notes

This is a POC (Proof of Concept) version with minimal features:
- No authentication
- No data persistence
- Basic error handling
- Basic UI styling
